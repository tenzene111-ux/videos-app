# Zhingkham 🇧🇹

A Bhutanese-inspired short video app — like TikTok, but the best of the best.

## Features

- **Vertical Video Feed** — Full-screen swipeable videos with auto-play
- **Discover** — Search and browse trending content by category
- **Create** — Camera interface with recording modes, filters, and effects
- **Inbox** — Notifications for likes, comments, follows, and mentions
- **Profile** — User stats, video grid, and settings

## Tech Stack

- React Native (Expo)
- React Navigation (bottom tabs)
- expo-av (video playback)

## Getting Started

```bash
cd zhingkham
npm install
npx expo start
```

Then scan the QR code with Expo Go on your phone, or press `w` for web.
